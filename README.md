# Taller #3 - Portafolio Profesional 2025

## Sintesis del Proyecto
Este proyecto es un portafolio presonal desarrollado en clase y bajo los parámetros y orientación del profesor del curso actual.

## Tecnologías
* HTML5 uso semántico:  header, main, sections, article... entre otros.
* CSS Moderno: implementación de Flexbox para interfaces.
* Responsive Design: Media queries para mejor visualización.

## Aprendizaje
- Contraste, experiencia de colores y tamaños.
- Manejo de navegación.
- Estructura de listas, manejo de repositorios y comandos git en VS Code.
